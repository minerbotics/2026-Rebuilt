# 2026-Rebuilt

## Python

This is our teams first year uses Python for our robot code. All previous years used Java. We are using the Command2 framework rather than MagicBot framework because CTRE uses the Command2 framework when generating the swerve code and we want to remain consistent. This also more closely resembles our Java code from seasons past. 

## Phoenix Tuner Swerve Generator

Our robot is using the SDS (Swerve Drive Specialties) MK5n swerve modules with the Kraken X60 drive motors. This allowed us to use the Phoenix Tuner X Swerve Project Generator.  Our robot is based on the kitbot, but in addition to upgrading the drivetrain to swerve, it uses CTRE motor controllers with Rev NEO motors for each of the other subsystems (fuel and elevator).
